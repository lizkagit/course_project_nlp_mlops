# Model Card: Comment Predictor

## Основная информация
- **Название модели**: Ridge Comment Predictor
- **Версия**: 1.0.0
- **Тип модели**: Регрессия
- **Задача**: Предсказание количества комментариев по тексту поста
 

## Назначение модели
Модель предсказывает, сколько комментариев получит пост в социальной сети на основе его текста.

### Варианты использования
- Анализ эффективности контента
- Прогнозирование вовлеченности аудитории

### Ограничения
- Работает только с русским текстом
- Максимальная длина текста: 10000 символов
- Не учитывает изображения, видео или время публикации


## Данные
### Используемые признаки
- `processed_text`: предобработанный текст поста (TF-IDF векторы)
- Признаки: униграммы и биграммы (1,2-граммы)
- Размерность: 5000 признаков

### Целевая переменная
- `comments_count`: количество комментариев (целое число)
